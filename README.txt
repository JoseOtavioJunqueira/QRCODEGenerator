Código gerado por José Otávio Junqueira Ramos para o projeto da diretoria TÉCNICA da SAECOMP - USP SÃO CARLOS

O código lê uma planilha do google sheets e analisa uma coluna de verificação de pagamento - OBS: a primeira linha da coluna deve contar "Conferiu Banco?"
Após isso, se a linha nao está vazia, o programa gera um QRCODE e envia para o email da pessoa.
Lembre-se de definir corretamente os parâmetros para que o programa corra de maneira correta!
